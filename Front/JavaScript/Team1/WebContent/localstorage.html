Local Storage

Web Storage API의 두 가지 방식 중 하나인 localStorage 입니다.
localStorage는 유효기간 없이 데이터를 저장하여 브라우저를 닫았다 열어도 데이터가 남아있으며, 다른 방식보다 저장 공간이 제일 큰 것이 특징입니다. (5MB)
